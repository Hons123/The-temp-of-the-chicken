# The-temp-of-the-chicken4.0
鸡鸡的温度4.0
此模块用于伪装CPU、GPU和电池的温度
默认无温度伪装，仅解除温度限制。
配置文件位于 /storage/emulated/0/Android/hons/鸡鸡温控.rc。
也可以点击模块的操作按钮进行开关。
